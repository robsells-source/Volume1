1903 King Edward VII, various frames

 

  1 c green
  3 c lilac
  3 c red
  4 c lilac on red
  4 c red
  4 c lilac
  8 c lilac on blue
  8 c blue

The 1 c, 3 c and 4 c designs have different palmtrees in the background, representing Penang, Singapore and Malacca. The 8 c has two krisses (a local dagger) in the design.
