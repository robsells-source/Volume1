1868 Queen Victoria facing the right, so-called 'Large Queens'

 

  1/2 c black
  1 c brown
  2 c green
  3 c red
  6 c brown
  12 1/2 c blue
  15 c lilac

All values exist with perforation 12. The values 1/2 c and 15 c also exist with perforation 11 1/2 x 12.
