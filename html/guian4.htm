Fiscal stamps

1869 Tree, inscription "INLAND REVENUE"


(Reduced sizes)

These stamps should be perforated. The values $1 black, $2 orange, $3 brown, $4 green, $5 grey, $6 brown, $7 lilac, $8 lilac, $8 blue, $9 yellow, $10 black, $12 orange, $16 blue, $20 red, $30 green and $40 red exist.

 

1865 Inscription 'Summary Jurisdiction British Guiana', large stamps

The following values exist: 12 c orange, 16 c green, 24 c black, 48 c lilac, 48 c red, 50 c brown, 60 c blue,72 orange 96 c lilac, 96 c grey and $1 yellow. Furthermore two provisional surcharges eixst: 50 c on 12 c yellow (1875) and 50 c on 60 c blue (1887).

 

Late 1800? "INLAND REVENUE THIS STAMP IMPLIES NO GOVERNMENT GUARANTEE"

  4 c duty (value 32 c and under) black and green
  8 c duty (value 64 c and under) black and green
  8 c duty (value 32 c and under) black and red
  24 c duty (value 96 c and under) black and red

These stamps are not listed in the Forbin guide.

 


This type of stamp (postage stamps with overprint "INLAND REVENUE") are listed under the postage stamps. Even forgeries exist of these stamps.

Tramway Stamp

'Le Timbre Poste' 362 (1893), page 18 lists a 'Tramways parcels delivery' stamp for Georgetown in the value of 2 c (issued around 1893). I have never seen this stamp.
